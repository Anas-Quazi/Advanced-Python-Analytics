## 🧠 Topics Covered

### 1. Python Basics
- Data types, variables, and operators
- Strings, lists, tuples, sets, and dictionaries
- Input/output and type conversions

### 2. Logic Building
- Conditional statements (`if`, `elif`, `else`)
- Loops (`for`, `while`) and comprehensions

### 3. Functions
- Defining and calling functions
- Arguments, default values, `*args`, `**kwargs`
- Lambda functions, closures, and decorators

### 4. Object-Oriented Programming (OOP)
- Classes and objects
- `__init__`, instance vs class attributes
- Inheritance, polymorphism, and encapsulation
- Dunder/magic methods

### 5. Libraries *(coming soon)*
- **NumPy** — arrays, vectorized operations, linear algebra
- **Pandas** — DataFrames, data wrangling, groupby, merging
- **Matplotlib** — line, bar, scatter, and custom plots
- **Seaborn** — statistical visualizations, heatmaps, pairplots
